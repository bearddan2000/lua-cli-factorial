# lua-cli-factorial

## Description
Demonstrates:
- calling local function
- string interpolation

## Tech stack
- lua

## Docker stack
- woahbase/alpine-lua:x86_64

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
